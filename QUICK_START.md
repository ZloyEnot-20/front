# 🚀 Quick Start Guide - Myfair Platform

## 30 секунд до первого теста

### 1️⃣ Запуск приложения
```bash
# Приложение уже готово к работе в v0
# Просто используйте Preview button
```

### 2️⃣ Вход в систему как Visitor
```
URL: /auth/login
Email: visitor@example.com
Password: password
[Нажмите "Вход"]
```

### 3️⃣ Регистрация на выставку (за 3 клика)
```
1. Главная страница → Карточка выставки → "Зарегистрироваться"
2. Форма → Введите Имя, Фамилию, Email, Телефон → "Далее"
3. Выберите город → [Система отправляет в Bitrix + генерирует QR]
4. Готово! QR код в личном кабинете
```

### 4️⃣ Просмотр QR-кода
```
URL: /participant
Вкладка: "Мои выставки (QR)"
✓ Видите QR-код и информацию регистрации
```

## 🔄 Процесс регистрации на выставку (6 шагов)

```
Шаг 1: Visitor видит выставку
        ↓
Шаг 2: Выбирает город (на форме)
        ↓
Шаг 3: Система отправляет ФИ, Email, Телефон в Bitrix
        ↓
Шаг 4: Формируется QR-код с данными регистрации
        ↓
Шаг 5: QR-код отображается в личном кабинете
        ↓
Шаг 6: Visitor получает письмо об успешной регистрации
```

## 📞 Что дальше?

### Для локальной разработки:
```bash
npm install
npm run dev
# Откройте http://localhost:3000
```

### Для интеграции реальных сервисов:
1. **Email:** Обновите `/lib/email-service.ts` → SendGrid/Mailgun
2. **Bitrix:** Обновите `/lib/email-service.ts` → Ваш API токен
3. **БД:** Замените `AdminContext` на реальную базу данных
4. **Аутентификация:** Добавьте JWT и сессии

## 🐛 Reporting Issues

Логи отправляются в консоль браузера (F12 → Console):
- Все `[v0]` логи показывают процесс
- Ошибки помечены как `[v0] Ошибка:`
